Welcome to DevExtreme Angular Template!

The application is build with:

  • Angular CLI
  • DevExtreme CLI
  • DevExtreme-Angular components

The application layouts are implemented using the Drawer component. Several component examples are also available in the Examples section. For the complete information on DevExtreme, check our documentation and demos.

You can do the following to create a custom application using this template as a base point:

  • Add new views
  • Switch theme
  • Create a Custom Theme
  • Apply the Additional Theme to a Custom Element
  • Apply Theme Colors to Custom Elements
  • Switch Layout

Please, read the Readme in devextreme-angular-template GitHub repo for more information.
