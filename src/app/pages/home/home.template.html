Welcome to My Application!

The application is build with:

  • Angular CLI version 6.0.0-rc.4
  • DevExtreme with DevExtreme-Angular extension

The application layout is implemented using the Drawer widget. Several widget examples are also available in the Examples section. For the complete information on DevExtreme, check our documentation and demos.

You can do the following to create a custom application using this template as a base point:

  • Modify existing views.

    The predefined views are located in the "app/pages" folder.

  • Add new views

    Use the appropriate command.

  • Move menu items

    The menu structure is specified in the "app/app-navigation.ts" file.
